\begin{frame}{Задачи}
В соответствии с поставленными целями в работе решаются следующие \textbf{задачи}:
\begin{itemize}
\item исследование операций классического алгоритма метода эллипсоидов на вычислительную сложность;
\item разработка программной реализации алгоритма с распараллеливанием наиболее длительных по времени операций;
\item обеспечение поддержки арифметики расширенной и произвольной точности;
\item демонстрация использования разработанного ПО для решения задачи оптимизации большой размерности;
\item проверка и тестирование разработанного ПО.
\end{itemize}
\end{frame}
