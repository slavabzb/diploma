\begin{frame}{Использование метода эллипсоидов}

\begin{block}{МЭ используется для решения следующей задачи}
На $E_n$ $(n\ge 1)$ определено векторное поле $g(x)$, $g(x)\in E_n$. Требуется найти точку $x^*$, такую, что $\left(g(x),x-x^*\right)\ge 0$ для всех $x\in E_n$. Предполагается, что $x^*$ существует и $g(x)\neq 0$ для $x\neq x^*$.
\end{block}

\begin{exampleblock}{}
К такой задаче сводятся задачи математического программирования:
\begin{itemize}
\item задача безусловной минимизации выпуклой функции,
\item общая задача выпуклого программирования,
\item задача о седловой точке выпукло-вогнутых функций.
\end{itemize}
\end{exampleblock}

\end{frame}
